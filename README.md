# Three.js 3D Model Sample — Millennium Falcon

Visor web sencillo hecho con **Three.js** para cargar y visualizar un modelo **glTF** del *Millennium Falcon* con controles de cámara (OrbitControls), iluminación y sombras.

---

## ✨ Qué incluye
- Render con **WebGLRenderer** y antialias.
- Cámara **PerspectiveCamera**.
- Controles **OrbitControls** (rotar y zoom; pan deshabilitado).
- Plano de piso con material estándar + sombras.
- Luz tipo **SpotLight** intensa para resaltar el modelo.
